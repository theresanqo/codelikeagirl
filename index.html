Lesson 1: Introduction

HTML

What is HTML?

HTML stands for 'Hypertext Markup Language'

'Hypertext' allows us to include hyperlinks, which allow us to navigate to other pieces of content with a click of a button.

A 'Markup Language' allows us to add tags to a plain text document so we can define structure and add meaning to text.

What are HTML elements?

HTML elements are the basic building blocks of HTML pages; they consist of content and opening/closing tags.

What are attributes?

Attributes allows us to add information to the element.

What is a DOCTYPE declaration?

HTML documents always begin with a DOCTYPE declaration, as they allow us to tell the browser which version of HTML the document is written in.

CSS

What is CSS?

CSS stands for Cascading Style Sheets. CSS allows us to change the presentation (such as layout, colour of text and backgrounds, fonts & size of HTML elements).
